How does this work?

A quick explanation of what this tool does under the hood.

The Problem

Normally, someone can only message you on WhatsApp if they've saved your number in their contacts first. This tool skips that step entirely.

How to use it

  1. Select your country code from the dropdown
  2. Enter your WhatsApp number without the leading zero
  3. Optionally type a pre-filled message
  4. Click Generate Link and share the result

The Link Format

The generated link follows WhatsApp's official URL scheme:

https://wa.me/{countrycode}{number}?text={message}

The message is encoded using URI encoding — for example, spaces become %20 so the URL stays valid.

Example

Country: Indonesia (+62), Number: 8123456789, Message: Hello!

https://wa.me/628123456789?text=Hello!
